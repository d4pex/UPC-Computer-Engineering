# Computer Engineering & Science Portfolio 🎓

**Universitat Politècnica de Catalunya (UPC - BarcelonaTech)**

This repository serves as a technical archive of the engineering projects developed during my B.S. in Computer Engineering. Unlike my [Security Portfolio](LINK_A_TU_OTRO_REPO), this collection focuses on the **underlying computer science foundations**: low-level architecture, operating system internals, algorithm optimization, and enterprise software architecture.

> **Security Perspective:** *You cannot secure what you do not understand.* This portfolio demonstrates deep knowledge of how systems are built "under the hood"—from memory management in C++ to TCP/IP packet structures—providing the necessary background for advanced Exploit Development, Reverse Engineering, and Red Teaming.

---

## 🗂️ Technical Domains

### 1. 🏗️ Software Engineering & Architecture
*Building scalable, maintainable, and robust software systems.*

* **`Software-Engineering-Management` (C++ / SQL)**
    * **Focus:** Enterprise Software Architecture (3-Tier), Design Patterns (Transaction Script, DTOs, Gateways).
    * **Security Relevance:** Deep understanding of Business Logic flaws, Session Management, and SQL Injection prevention via architectural design.
* **`Agile-FullStack-Development-NodeJS` (Node.js / Express / JS)**
    * **Focus:** Modern Web Development, REST APIs, and Scrum Methodology.
    * **Security Relevance:** Hands-on experience with Middleware logic, API authentication (JWT), and common web vulnerabilities (Mass Assignment, IDOR).
* **`Data-Structures-Algorithms` (C++ / STL)**
    * **Focus:** Algorithmic complexity ($O(n)$), Memory Management, and Data Structures (Tries, Hash Tables).
    * **Security Relevance:** Understanding buffer handling and memory allocation is crucial for Exploit Development.

### 2. ⚙️ Low-Level & Systems Programming
*Direct interaction with Hardware, Kernel, and Memory.*

* **`Operating-Systems-Internals` (C / Linux API)**
    * **Focus:** System Calls (`fork`, `exec`, `pipe`), Concurrency, and Signal handling.
    * **Security Relevance:** The foundation of Malware Development and Evasion techniques using Linux internals.
* **`Computer-Architecture-Assembly` (Assembly)**
    * **Focus:** Processor architecture, Interrupts, Stack management, and I/O.
    * **Security Relevance:** Essential for Reverse Engineering and Shellcode development.
* **`Parallel-Computing-HPC` (C / OpenMP)**
    * **Focus:** High-Performance Computing, Race Conditions, and Thread Synchronization.
    * **Security Relevance:** Understanding Race Conditions is key for identifying Time-of-Check to Time-of-Use (TOCTOU) vulnerabilities.
* **`Computer-Graphics-OpenGL-Qt` (C++ / OpenGL)**
    * **Focus:** Graphics Pipeline, Shaders (GLSL), and Matrix Math.
    * **Security Relevance:** Low-level memory manipulation and GPU interaction concepts used in Game Hacking.

### 3. 🌐 Networks & Distributed Systems
*Infrastructure, Protocols, and Connectivity.*

* **`Computer-Networks-Protocols` (Wireshark / Packet Analysis)**
    * **Focus:** Deep inspection of the TCP/IP stack (ARP, ICMP, TCP, HTTP).
    * **Security Relevance:** Manual packet analysis for Sniffing and Traffic Analysis.
* **`Internet-Technologies-Infrastructure` (Mikrotik / Routing)**
    * **Focus:** Network Design (VLSM), Routing (Static/Dynamic), and Firewall Configuration (Stateful).
    * **Security Relevance:** Network Mapping, Pivoting, and Firewall Evasion techniques.
* **`Distributed-Systems-AI` (Erlang / Python)**
    * **Focus:** Concurrency, Message Passing, and Consensus Algorithms.
    * **Security Relevance:** Analysis of distributed logic and fault tolerance in large-scale systems.
* **`Linux-System-Administration-Scripting` (Bash / Python)**
    * **Focus:** System Hardening, Automation, and Service Management (Systemd).
    * **Security Relevance:** Persistence mechanisms, Privilege Escalation contexts, and Living-off-the-Land (LotL) scripting.

### 4. 🧠 Artificial Intelligence & Logic
*Algorithms, Heuristics, and Decision Making.*

* **`Artificial-Intelligence-Algorithms` (Java)**
    * **Focus:** Search Strategies (A*, BFS, DFS), Game Theory (Minimax), and Heuristics.
    * **Security Relevance:** Graph theory concepts applied to Attack Path Mapping (e.g., BloodHound) and automated decision-making logic.
